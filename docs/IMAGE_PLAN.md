# Brightspace Documentation Image Plan

> A comprehensive plan for screenshots and images needed across all documentation guides.
> This document maps each image to the Brightspace navigation path required to capture it.

## Image Specifications

- **Format**: PNG (lossless)
- **Max Width**: 600px
- **Max File Size**: 100KB target, 200KB max
- **Annotations**: Red boxes/circles, numbered callouts (1), (2), (3)
- **Naming Convention**: `category-guide-description.png` (e.g., `getting-started-interface-minibar.png`)
- **Storage Location**: `/public/images/docs/`

---

## Priority Legend

| Priority | Description |
|----------|-------------|
| **P1 - Critical** | Essential for understanding, highly visual concepts |
| **P2 - Important** | Significantly improves comprehension |
| **P3 - Nice to Have** | Helpful but text description works |
